# API Documents

## 运行项目 ##
> 在域名下面访问 http://domain/index.html?module=demo

## 生成文档 ##

### 全局安装jsdoc ###
> npm install -g jsdoc

### 验证是否成功 ###
> jsdoc -v //任意地方执行此代码验证jsdoc是否安装成功

### 运行文档生成命令 ###
> jsdoc -c .jsdoc3 //在根目录下执行命令

## 目录说明 ##
- app: 子系统模块(子应用)
- assets: 基础框架
- docs: 基础框架文档
- templates: jsdoc3文档模板
- index.html: 应用首页面
- login.html: 应用登录页面
